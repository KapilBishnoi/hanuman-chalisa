# Hanuman Chalisa

श्रीगुरु चरन सरोज रज निज मनु मुकुरु सुधारि ।\
बरनउँ रघुबर बिमल जसु जो दायकु फल चारि ॥\
बुद्धिहीन तनु जानिके, सुमिरौं पवन कुमार ।\
बल बुधि विद्या देहु मोहि, हरहु कलेश विकार ॥
